INTRODUÇÃO

A linguagem C é uma linguagem de programação de alto nível, sendo muito influente no aprendizado de lógica de programação e conhecimentos práticos de desenvolvimento para estudantes de universidades e na construção de grandes softwares de sistemas embarcados de GPS de automóveis ou em sistemas operacionais como o Linux.

A linguagem C foi criada por Dennis Ritchie na década de 1970 tendo como objetivo a criação do sistema operacional UNIX. Porém, rapidamente se tornou uma linguagem muito popular e utilizada em outros grandes sistemas pelo fato de ser simples e muito eficiente em situações onde o processamento e espaço de memória é muito baixo, esse fato torna até mesmo hardware muito inferiores utilizáveis. Sua usabilidade simples e funcionalidades poderosas, a torna uma linguagem ideal a ser explorada por aqueles que querem aprender sobre o mundo da programação.

Nesta tese, iremos abordar os pontos principais das funcionalidades e usabilidades na linguagem com objetivo de contribuir com a aprendizagem do leitor sobre os conceitos básicos da linguagem de programação C, assim podendo desenvolver novas práticas com a programação.

DESENVOLVIMENTO

As bibliotecas usadas nos códigos da linguagem C são de suma importância para o bom funcionamento do programa. Cada uma desempenha uma função específica, ajudando o programador a melhorar sua eficiência e simplificar comandos. Apesar de não ser necessário utilizar bibliotecas que não irão ajudar, colocá-las não quebram o projeto. Porém o programa pode não funcionar caso não sejam incluídas as bibliotecas corretas.

Dentro do processamento de dados é constituído em três etapas para que o sistema execute certo. Se inicia com a entrada de dados, onde o usuário insere as informações básicas como um valor, uma palavra, entre outros, essas informações são armazenadas dentro de uma variável. Na segunda etapa vem o processamento de dados, onde é processado e executado essas informações declaradas pelo usuário, que no entanto a última etapa é a saída de dados que nos proporciona um resultado final desta operação.

BIBLIOTECAS

As bibliotecas usadas na linguagem C possuem muita importância na hora de produzir um código. Elas dão um ampla gama de funções pré-definidas, fazendo com que o programador use-as para facilitar a execução de algumas tarefas. Para conseguir usar uma biblioteca, deve-se digitar ‘#include’, assim ela funcionará no código.

Além disso, alguns comandos só funcionam corretamente se a biblioteca certa estiver sendo utilizada. Ainda assim, usar mais bibliotecas do que será necessário não irá prejudicar seu programa, porém sua função será inútil. Aqui estão breves introduções de algumas das principais bibliotecas usadas na linguagem C :

stdio.h

A biblioteca ‘stdio.h’ é útil para realizar a entrada e saída de dados num programa, a exemplo, os comandos “printf” (saída de dados) e “scanf”(entrada de dados). Seu nome significa, em português, entrada e saída padronizadas.

stdlib.h

A biblioteca ‘stdlib.h’ usa alocação e desalocação de memória, além de poder converter números que estão representados em uma string para dados que equivalem a números.

string.h

A biblioteca ‘string.h’, como o próprio nome já menciona, é usada para operar strings que são cadeias de caracteres que sejam 2 ou mais letras e símbolos especiais. Ela pode 3 copiar caracteres, compará-los e até procurar por substrings. Cada uma de suas funções possuem finalidades diferentes em quaisquer cenários.

math.h

A biblioteca ‘math.h’ é utilizada para que operações matemáticas complexas sejam feitas com mais facilidade. Pode-se fazer cálculos com funções trigonométricas, raízes quadradas, logaritmos, entre outros. Usando essa biblioteca os valores retornam como tipo “double”.

PROCESSAMENTO DE DADOS

Processamento de dados em linguagem C trata-se de como manipular informações em um programa escrito em C. Com isso conseguimos fazer a leitura de dados de arquivos, cálculos matemáticos, composições de informações e exibição de resultados. Para conseguirmos executar corretamente esse processamento temos que realizar três etapas que são elas: Entrada de dados, processamento de dados e saída de dados. Na etapa de entrada de dados precisamos declarar as nossas variáveis. As variáveis em si servem para armazenar informações, como números, textos, caracteres e outros tipos de dados. Na etapa de processamento de dados o sistema vai avaliar os dados inseridos, e na última etapa que é a saída de dados ele vai nos informar o resultado que solicitamos ao sistema.

Por exemplo: Você pode solicitar para que o usuário insira dois valores e com esses dois valores seja feito uma soma. Neste caso você vai criar três variáveis inteiras A, B e soma, primeiro você vai solicitar para que o usuário insira os valores em A e B, depois dentro do printf você vai declarar que soma=A+B, e por fim você vai pedir para o programa informar este resultado na tela.

Vamos supor que o usuário informou no valor A = 10 e no valor B = 20, no entanto esse código exibirá o resultado “A soma dos valores é igual: 30” na tela. Simplificando o processamento de dados é a tarefa que o programa executa com os dados que recebe. No código informado acima como exemplo a entrada de dados são os valores, o processamento de dados é a soma destes valores, e a saída de dados é o resultado da soma dos valores.

ENTRADA DE DADOS

A entrada de dados em linguagem c é usada nas bibliotecas, mas especificamente nas funções dela , como scanf e fgets, , para receber informações dos dispositivos ou de algum usuário, recebendo arquivos também, a principal função é scanf, para usar ela é necessário definir a string de formato e as variáveis que será onde os dados vão ser armazenados, scanf executa as mesmas regras que a função printf, exemplo:"%d %f %s", resumindo a entrada de dados Tem por função efetuar a leitura de dados de uma fonte externa. Vou mostrar abaixo as principais funções de entrada de dados em c:

SCANF:

a função dele é ler o dado digitado pelo usuário, toda vez que acionado esse comando o programa irá esperar um dado vindo do teclado, que quando ocorrer será armazenado em uma variável.

GETCHAR:

Já o getchar é usado para ler um caracter do teclado, além de ler ele faz a função de converter para int.

FGETS:

serve para ler uma linha de texto inteira, e também espaço em branco e arquivos, muito útil para ler strings completas.
